Return the dataset of shakespeare, subreddit, twitter, or synthetic.

Parameters

Returns

FL dataset dict, with client_id as key.

Note

load_nlp_dataset() will return a dict as shown below:

` {'client_id': {'train': dataset, 'test': dataset, 'val': dataset}} `
